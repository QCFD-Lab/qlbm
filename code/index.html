Internal Documentation#

qlbm is made up of 4 main modules. Together, the Base Classes, Amplitude-Based Circuits, and QLGA Circuits module handle the parameterized creation of quantum circuits that compose QBMs. The Lattices and Geometry module parses external information into quantum registers and provides uniform interfaces for underlying algorithms. The Infrastructure module integrates the quantum components with Tket, Qiskit, and Qulacs transpilers and runners. The Other Tools module contains miscellaneous utilities.
